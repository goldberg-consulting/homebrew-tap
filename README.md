# Goldberg Consulting Homebrew Tap

Homebrew formulae and casks for tools by [Goldberg Consulting](https://github.com/goldberg-consulting).

## Install

```bash
brew tap goldberg-consulting/tap
```

## Available casks

| Cask | Description |
|------|-------------|
| `tidaldrift` | Menu-bar Mac utility for discovering, connecting to, and streaming between Macs |
| `distribute-metal` | macOS menu bar app for distributed PyTorch training on Apple Silicon using Metal and MCCL |

### TidalDrift

```bash
brew install goldberg-consulting/tap/tidaldrift
```

Requires macOS 13+ (Ventura).

### DistributeMetal

```bash
brew install goldberg-consulting/tap/distribute-metal
```

Requires Apple Silicon (M1+) and macOS 14+ (Sonoma).
